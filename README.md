## Getting Started

#### リポジトリのクローン
```shell
git clone https://github.com/iseruuuuu/wallet_connect_app.git
```

#### fvm のインストール

```
fvm use 3.16.5
```

#### 注意事項
- `projectID`については、各自Wallet Connectにサインインし、取得してください。

```dart
//lib/constants/secret_key.dart
const projectID = 'プロジェクトID';
```
